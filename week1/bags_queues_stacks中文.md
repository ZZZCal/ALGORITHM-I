# API(applicaiton programing interface)
## API结构：
>> 无参数构造函数\
>> 向collection中添加元素的方法\
>> 检测collection是否为空的方法\
>> 返回collection大小的方法\

## 泛型(Generics)
>> 也叫参数化类型
>> 可以用抽象数据类型来存储任意类型的数据

## 自动装箱(Autoboxing)
>> 参数类型必须被实例化为引用类型(reference type)
>> 泛型代码可以处理原始数据类型

## 可迭代的集合类型(Iterable collections)
>>  很多时候我们需要迭代访问集合中的所有元素
>> 一级语言特性(first-class status)

## 背包(bags)
>> 不支持从中
